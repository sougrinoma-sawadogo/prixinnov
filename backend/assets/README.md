# Assets Directory

## Armoiries du Burkina Faso

Placez l'image des armoiries du Burkina Faso dans ce dossier avec le nom suivant :

- **Nom du fichier** : `armoiries.png`
- **Format recommandé** : PNG avec fond transparent
- **Taille recommandée** : 200x200 pixels minimum (sera redimensionnée automatiquement dans le PDF)

Le fichier doit être placé à : `backend/assets/armoiries.png`

Si l'image n'est pas trouvée, un placeholder sera affiché dans le PDF.
